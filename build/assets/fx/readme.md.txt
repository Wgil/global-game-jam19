# hit - yellow.png

Frame by frame animations! :D PNG, Atlas format.

Just a single animation of a "hit" vfx.

It's a Sprite Sheet in which each frame is a 1024 x 1024 size image (4096 x 4096 total). 16 frames.

PNG format with transparency.

# smoke
